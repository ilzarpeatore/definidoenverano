## Technical Implementation Notes

### Database Schema Changes Needed:
```sql
-- Add to recommendations table
ALTER TABLE recommendations ADD COLUMN (
  resource_type ENUM('pdf', 'video', 'audio', 'checklist'),
  resource_url VARCHAR(500),
  resource_title VARCHAR(255),
  resource_duration INT, -- in minutes
  download_count INT DEFAULT 0
);

-- Track user downloads
CREATE TABLE user_downloads (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT,
  recommendation_id INT,
  downloaded_at TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (recommendation_id) REFERENCES recommendations(id)
);
```

### UI Component Changes:
- Update RecommendationCard to show resource type badge
- Add download button with analytics tracking
- Show resource duration/size
- Add "Resources Downloaded" counter
- Create resource library page
